# Juego de Memoria - Proyecto Web

## Descripción

Este proyecto es un **Juego de Memoria** implementado en **HTML**, **CSS** y **JavaScript**. El objetivo del juego es descubrir pares de tarjetas con íconos que se encuentran ocultos en el tablero. El juego se basa en la clásica dinámica de voltear tarjetas y encontrar sus pares. El diseño de las cartas y la interfaz se ha actualizado para tener un estilo moderno con tonos de **celeste/azul pastel**.

## Tecnologías utilizadas

- **HTML**: Estructura del contenido.
- **CSS**: Estilos y diseño visual, con énfasis en un diseño moderno y limpio.
- **JavaScript**: Lógica del juego, manipulación de eventos y control de interacciones.

## Características del Juego

1. **Barajar las tarjetas**: Las tarjetas con los íconos son barajadas aleatoriamente en cada partida.
2. **Descubrimiento de tarjetas**: El jugador hace clic en las tarjetas para descubrirlas y comparar los pares.
3. **Contador de movimientos**: Lleva un registro de los movimientos realizados por el jugador.
4. **Panel de Información**: Muestra un panel con información relevante, como el número de movimientos y la cantidad de pares encontrados.
5. **Finalización del juego**: Cuando todos los pares se encuentran, el juego se detiene y se muestra un mensaje de éxito.
6. **Estilo moderno**: El diseño utiliza colores de fondo blanco y cartas con gradientes de azul pastel y celeste, ofreciendo un ambiente limpio y agradable.
